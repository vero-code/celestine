Product Tour: From Space to Earth

An immersive Mission Control interface connecting cosmic discoveries to real-world Earth locations.

The Problem We're Solving

Static Education

STEM education remains abstract. NASA releases terabytes of data, but there's a disconnect. Students see Mars as a distant red sphere, unable to grasp the scale or geological reality of its surface.

Engagement Gap

Existing tools lack interactivity. High-quality simulators are too complex for casual learners, creating a gap where the universe feels inaccessible and irrelevant to daily life.

How It Works

STEP 01

Explore the Cosmos

Interactive 3D Solar System built with React Three Fiber. Users navigate freely or input commands like "Take me to Mars" to initiate a landing sequence.

STEP 02

AI Agentic Reasoning

Powered by Gemini 2.5. When you ask about a feature (e.g., "Olympus Mons"), the AI doesn't just recite factsâ€”it reasons about geological context, formulates a search strategy, and autonomously executes a tool call to the Google Maps Places API to find a terrestrial equivalent (e.g., Mauna Kea, Hawaii).

Validation: Large Language Models can effectively orchestrate geospatial APIs.

STEP 03

The Earth Analogue Loop

The system autonomously finds a terrestrial equivalent (e.g., Mauna Kea) using Google Places API and transports you there via Photorealistic 3D Tiles.

Key Technologies

Multi-Agent Orchestration

Custom Python backend where specialized agents handle navigation, scientific queries, and tool use.

Immersive Visualization

Seamless integration of NASA planetary textures and Google Maps Platform Photorealistic 3D Tiles.

Voice & Video Co-Pilot

Integration with ElevenLabs and Tavus for a lifelike, conversational guide experience.
